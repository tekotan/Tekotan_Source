Image Manipulation Project

Duration: Febuary 25, 2019 - March 28, 2019
Partner: Andrea Jia Platform: Python
Objective: To develop an image manipulation feature using Python and other APIs
Details: For the image manipulation project we had develop and design an image manipulation feature using python APIs and other modules. The original project provided 3 guided tasks, all involving a client bringing a task. We however decided to develop something more complex, and opt for Task 4, which was the development option. We decided to implement a style transfer system described in the paper A Neural Algorithm of Artistic Style , which would transfer the style of one image onto the content of another image. Since I had prior experience in reading and understanding algorithms in scientific papers, I headed the algorithm implementation while Andrea focused on the user interface and the documentation of the project. Some challenges that arose was in the implementation of the algorithm was that some of the techniques were too exhaustive for the resources available to us, in which case I used prior knowledge of Deep Learning to substitute the techniques with other viable options. Another challenge that arose was figuring out the best way to provide the user with choices in the user interface. We solved this challenge by taking the opinion of our classmates of what they want in a user interface, and using their opinons to build one. This project taught me about how to read and comprehend a scientific paper that outlines an algorithm, and how to use mathematics to understand a computer algorithm. Some soft skills I learned was how to explain an algorithm to someone that has no prior experience on the subject and it's background, something that is a very important skill for the professional world. The documentation for this project can be found here, and for the techinical details and paper summary see my blog on this project here.

Objective: To test the best neural network type for text based classifications
Details: For this project, I wanted to determine the best network architecture for text based classifications. I used the most popular Deep Learning techniques, such as Recurrent Neural Networks, Convolutional Neural Networks, and Densely Connected Layers. Because of my technology limitation of having a budget laptop, I chose the simpler task of classifying the sentiment of the text as opposed to something like text autocompletion or a translator. The main skills I learned during this process was how to stay focused on a task despite hiccups, and real time experience on the process of building an algorithm using tensorflow. Credit Card Fraud Detection

Duration: October 12, 2018 - December 10, 2018
Partner: My Father Platform: Python 3.6
Objective: To build a system that detects fraudulent transactions in a parking garage
Details: For this project, my father and I got a project proposal to use deep learning to detect fraudulent parking ticket transactions. This is a type of problem called anomaly detection that deals with data that has a dramatic class split. In this situation, the number of fraudulent transactions is greatly outnumbered by the normal, nonfraudulent transactions in the dataset. This is the primary challenge of this project. This challenge was overcome by 2 means, one empirical, and one means was proven through internet research. The empirical means used was oversampling the fraud data to increase it's presence in the training data. The second means was to use an autoencoder to extract the features of the data and learn the data's configuration. The use of an autoencoder allows the model to discard the data that doesn't help in identifying the class and to emphasize the deciding data points. The main skiils I learned in this project was how to develop in a way for a non technical client to use, and how to effectively explain a feature in a way that is impressive.
